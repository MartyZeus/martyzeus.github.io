Email Engine

Goal

I would like to build an Email engine with to automate the building and sending of emails. This will be programmed in JavaScript using Node.js as the runtime enviroment, Microsoft SQL Server as the database, and SMTP2GO as the SMTP relay.

JavaScript Build

The project will injest records from the SQL Server, restructure the rows into JSON, utilize Handlebars.js to build emails based on defined rules, and then perform HTTPS POSTs to the SMTP2GO's email deployment API.

Tech Stack
